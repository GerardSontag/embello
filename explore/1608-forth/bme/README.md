BME280 sensor explorations using the [Olimexino-STM32][O] board.

Only I2C and LEDs are used here, see `../oxs` sibling folder for other pinouts:

    D5      PB6     SCL1
    D9      PB7     SDA1

    D3      PA1     yellow LED
    D13     PA5     green LED

   [O]: https://www.olimex.com/Products/Duino/STM32/OLIMEXINO-STM32/
